# Круголёт

**Без потери времени**

Кроссплатформенное приложение на Godot для контроля домашних заданий и управлением временем, созданное для учителей и учеников.

## Основные возможности:
- Единый календарь для школы и доп. активностей
- Одна платформа для ДЗ и пройденных тем
- встроенный ИИ для приоритизации задач и оптимизирования сбора ДЗ

## Технологии:
- Godot Engine
- SQLite
- Python
